What is pubrunner?

PubRunner is a easy-to-use framework that keeps text mining tools running on the latest publications.

That means the latest results without the fuss!

Show Me Results Add My Tool

This is in Alpha. So expect issues.

What's the problem?

New text mining tools are developed and published regularly. But often they are only run once and new results are never produced on newer publications. The results soon become stale and therefore useless for other researchers. Let's change that!

What tools are using it?

At the moment, only a small set of test tools are being run. But soon more will be added! Check out the Results page for the list of the tools and their latest results.

Can I add my tool?

Yes please! It's pretty basic. You just need to install PubRunner on the machine where you do your text mining and it'll do the rest. Check out the tutorial in the Github readme

What is PubRunner?

PubRunner is a basic framework to help you run your text mining tool on a regular basis with the latest Pubmed abstracts.
